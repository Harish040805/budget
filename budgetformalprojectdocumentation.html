Budget Management - Project Features

Formal Documentation

1. Green AI - Intelligent Login System

The Green AI module implements a secure, dynamic login page developed using React JS and enhanced with artificial intelligence. Unlike a standard login form, this system features modern authentication that automatically submits once all required user details have been entered. It also includes AI-driven validation and CAPTCHA verification. The system performs a dual role, acting as both a login and a sign-up page. If a user is new, it automatically stores their data when the fields are filled, functioning as a sign-up page. If an existing user fills the form, it functions as a login page. The AI monitors login attempts for unusual patterns, thereby improving security and reducing the risk of automated attacks. CAPTCHA integration adds an additional safeguard by distinguishing between human users and automated scripts. The interface is designed with a green theme and includes a glitter animation across the site, keeping the visual appearance consistent with the overall project identity. Additionally, the system features a circular password strength meter—represented with a dynamic progress circle and a shield icon—that updates in real time as the user enters their password and other login details. This meter visually indicates the strength and completeness of the form inputs, providing immediate feedback to users and enhancing both usability and security.

2. Budget Management Site

The Budget Management Site is designed to help users track and manage their daily expenditures on products and services. The interface allows users to input purchase details and maintains the data locally, ensuring that expenses can be reviewed at any time without requiring an internet connection. The site makes expense recording quick and user-friendly. The calculations are dynamically performed for the price entered by the user. It works similar to Microsoft Excel, but additionally, the dynamic calculation and autosave functionality ensures that the user-entered information is never lost accidentally. Creating a new row and column is done by dynamic DOM manipulation in JavaScript.

3. Income Management Site

The Income Management Site provides a structured way for users to categorize and allocate their income across multiple essential and optional categories. Input fields are restricted to numeric values, ensuring that data remains consistent and accurate. The supported categories cover common household and personal financial needs:

  • Rent
  • Utilities
  • Grocery
  • Transportation
  • Insurance
  • Entertainment
  • Health
  • Education
  • Savings
  • Other

This categorization not only simplifies record-keeping but also enables users to analyze their spending patterns and plan future budgets more effectively. When the user enters their total salary, the site facilitates salary allocation across categories while ensuring that the allocations do not exceed the total salary. If the entered amounts surpass the available salary, the system dynamically adjusts the maximum allowable value, thereby preventing users from exceeding the defined input limit.

4. Profit and Loss Calculator

The Profit and Loss Calculator is designed to help users dynamically determine profit or loss values, along with their corresponding percentages. By entering the Cost Price (CP) and Selling Price (SP), the system instantly computes whether the transaction resulted in a profit or a loss.

The calculator provides the following outputs:

  • Profit Amount - when the Selling Price exceeds the Cost Price.
  • Loss Amount - when the Cost Price exceeds the Selling Price.
  • Profit Percentage - calculated as (Profit ÷ Cost Price) × 100.
  • Loss Percentage calculated as (Loss ÷ Cost Price) × 100.

This tool is especially useful for students, businesses, and individuals to quickly analyze financial transactions without manual calculations. The dynamic computation ensures accuracy and saves time, making it a practical resource for day-to-day use in business mathematics.

5. Integration of Local Database with Autosave

Both the Budget Site and the Income Site are equipped with a local database mechanism. This design choice allows data to be stored directly in the user's browser, eliminating the dependency on external servers. The autosave feature records every change instantly, preventing data loss. Additionally, a clear button is provided, giving users full control to remove all stored information from the local database whenever required. This balance of persistence and flexibility makes the system both reliable and user-centric.

6. Download Functionality

The Budget Site provides a download feature that allows users to export the currently entered data sheet directly to their local storage. The exported file can be viewed either as an Excel spreadsheet or as a formatted budget table, depending on the user's preference. While the exported Excel sheet serves as a static record without autosave, dynamic calculations, or automatic salary segregation, the web application itself continues to support these advanced features. This ensures that users benefit from both the portability of offline records and the enhanced functionality of the online system.

7. Color Theme

The project adopts a green and white color theme as its primary design identity. Green was chosen to symbolize growth, balance, and sustainability, aligning well with the purpose of managing financial health. The consistent use of green across all modules ensures visual harmony and a professional appearance.

9. Splash Screens

Each main module of the project is introduced with a splash screen. The splash screens serve as entry points, displaying the title and theme of the respective module before the main interface loads. This enhances user experience by providing a smooth and professional transition into the application. Additionally, a custom logo related to economics and budgeting is being designed using Figma, exported and integrated into all splash screens, ensuring visual consistency and reinforcing the theme of budget management across the site.

10. Artificial Intelligence Integration (Speech AI)

Google's Gemini AI is integrated throughout the project to enhance functionality and user experience. It provides intelligent validation, advanced input assistance, and robust security features. In the Green AI login system, the AI detects unusual login patterns and automates CAPTCHA verification. Within the Budget and Income sections, it offers guidance on optimal salary allocation, flags potential overspending, and delivers personalized financial recommendations. Accessible via a draggable floating bot button, the AI assistant can speak responses, provide contextual support, and address user queries, ensuring the platform adapts dynamically to user needs. By combining AI-driven decision support with local database persistence, the system delivers both intelligence and reliability in personal finance management.

11. Live Time and Date

This feature displays the current time and date dynamically on the interface. Users can toggle a dedicated panel to view real-time updates of hours, minutes, seconds, day of the week, and full date. The time is updated every second, ensuring accurate tracking. This is implemented using JavaScript's Date object and setTimeout to refresh the display continuously. The panel is designed with a green-themed gradient and semi-transparent background for readability and visual consistency with the overall project theme.

12. Simple Calculator

The Simple Calculator provides users with basic arithmetic operations (addition, subtraction, multiplication, division) directly within the application. It includes a numeric display and a grid of buttons for numbers and operators. Users can input numbers, perform calculations, and clear the display as needed. The calculator uses JavaScript to handle input, calculation, and error detection, ensuring that invalid operations are caught and reported. This feature is accessible via a floating toggle button and appears as a centered panel, styled with green-themed gradients, rounded corners, and shadows to maintain consistency with the project's visual identity.
